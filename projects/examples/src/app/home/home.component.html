UI Components

This website showcases components used in VMware Cloud Director's UI and that can be used by plugin developers with two main goals

  • Shared user experience - plugin developers get the same experience that will remain consistent with VCD's UI even as it evolves
  • Shared code - reuse core functionality that we've found helpful within VCD's UI

Installation

npm install @vcd/ui-components

For further info and source code, visit our github page.

Using this documentation

The left menu contains links to reusable components. Clicking them will take you to a page that shows three tabs:

Description/ Class level description of the component
API @Input and @Output of components
Examples Live examples with which can interact, view source code, or even launch in a stackblitz editor to help you play with its API
